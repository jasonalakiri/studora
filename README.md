A simple study planner made with HTML, CSS and JavaScript.
No installs. Just open index.html in your browser.

## Files
  index.html  the page.
  smartstudy.css   all the styles
  smartstudy.js   all the logic
  imgs/green image
## What it does/ Features
- Add, edit, delete tasks
- Mark tasks as done
- Progress bar showing how many tasks are done
- Pomodoro timer (25 min study, 5 min break)
- Light and dark mode
- Everything saves automatically with localStorage
